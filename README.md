# Design-and-Development-of-the-College-Event-Webpages
A dynamic University Events Web Application built using HTML, CSS, and JavaScript. It showcases events like hackathons, symposiums, and health camps with detailed pages, registration forms, OTP verification, participant tracking, and admin tools for managing, analyzing, and exporting event results efficiently. Includes responsive design and goodUX.

🎓 University Events Web Application

A dynamic and interactive web application designed to showcase and manage university events such as Hackathons, Symposiums, and Health Camps. Built using HTML, CSS, and JavaScript, this project provides event details, registration systems, and participant management features.

🚀 Features
📅 Event listing with attractive UI
📄 Detailed pages for each event
📝 Registration forms for participants
🔐 OTP verification system (demo)
👥 Participant data storage (frontend-based)
🏆 Admin panel to select winners
📊 Automatic score calculation (Hackathon)
📥 Download participant data as CSV
🎨 Responsive design with animations
